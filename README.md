# Andrew Loniewski Portfolio

Personal portfolio built with Vite, React, TypeScript, and Tailwind CSS.

## Quick Start

```sh
# Install dependencies
npm i

# Start dev server
npm run dev
```

## Scripts

```sh
npm run dev
npm run build
npm run preview
npm run lint
npm run test
```

## Project Structure

- `src/` app code
- `public/` static assets
- `src/data/projects.ts` project metadata

## Notes

- Videos are in `public/videos/`
- Resume file is `public/resume.pdf`
